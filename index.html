Unofficial guidance on various topics by Social Science Data Editors

These pages provide guidance to authors on creating replicable data and program archives. The data and reproducibility editors from multiple journals who have helped to create this guidance have cumulatively reviewed several thousand replication packages since 2018.

You can navigate this site by searching (top right), by keywords (cloud to the right), or by starting with the guidance for authors.

Other Information

Other materials, sometimes referenced within this guidance, can be found elsewhere on this website (see also the top menu on the right):

  • Data and Code Availability Standard: A standard for sharing research code and data, endorsed by leading journals in social sciences.
  • Template README: The template provided on this website is in a form that follows best practices as defined by a number of data editors at social science journals.
  • Discussion forum for (Data) Editors: The data and reproducibility editors meet regularly. More information can be found on the main website.
